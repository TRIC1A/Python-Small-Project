1. Parentheses()
-> Used for Tuples and Functions
-> Functions: They are use to "call" a function or pass information(arguments) into it.
-> Tuples: A data structure that is immutable(cannot be changed once created).

2. Square Brackets[]
-> Used for Lists and Indexing.
-> Lists: The most common way to store a collection of items that can be changed(mutable).
    > Usage: shopping_list = ["Eggs", "Milk", "Bread"]
-> Indexing/Slicing: Used to grab a specific item out of a list or string by its position.
    >Usage: first_item = shopping_list[0] (Remember, coding starts at 0!)

3. Curly Braces {}
-> Used for Dictionaries, Sets, and f-strings
-> Dictionaries: Store data in "Key: Value" pairs. Great for labeled data.
    > Usage: user = {"name": "Tricia",
                     "age": 23}
-> Sets: Store unique items and automatically remove duplicates.
    > Usage: unique_numbers = {1,2,2,3} (Result is {1,2,3})
-> f-strings: Used inside a string to "plug-in" a variable.
    >Usage: print(f"Hello {name}")
